2d canvas api - First foray into 3d rendering. Projects a scene to a (virtual) image plane and uses the 2d canvas api to draw it with lines and fills. Runs pretty smoothly but it's very limited.
full rasterization algorithm - Lib-free & minimal canvas api implementation of a rasterizer running on the js main thread. Looks decent, but barely manages a usable framerate.
WebGL - Using what I've learned so far with an actual graphics api that runs on the gpu.
Raytracer - Mostly an exercise in copying/transposing an example from c++ to js. Not interactive, and takes a few seconds to render. Will get more into raytracing later.
